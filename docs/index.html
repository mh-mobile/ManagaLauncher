マンガ曜日

曜日ごとのマンガをワンタップで

主な機能

曜日ごとに管理

週間連載のマンガを曜日ごとに登録。今日読むマンガがひと目でわかります。

ワンタップで起動

タップするだけでマンガサイトやアプリをすぐに開けます。

既読・未読管理

更新曜日に自動で未読に戻り、読んだら既読に。読み忘れを防ぎます。

更新頻度の設定

毎週・隔週・月1回など、作品ごとの更新頻度に対応。次の更新日も管理できます。

キャッチアップ

未読マンガをカードをスワイプしてサクサク消化。既読・あとでをすばやく振り分け。

休載・完結に対応

休載中や完結した作品を専用タブで管理。連載再開時もスムーズに戻せます。

読書ヒートマップ

GitHub風のヒートマップで読書の記録を可視化。日々の読書量がひと目でわかります。

ウィジェット対応

ホーム画面やロック画面から今日のマンガを確認・起動できます。

2つの表示モード

リスト表示とPinterest風グリッド表示を切り替えて、お好みのスタイルで閲覧。

iCloud同期

iPhone・iPad・Mac間でデータを自動同期。どのデバイスからでも同じマンガにアクセス。

バックアップ

JSON形式でのエクスポート・インポートに対応。データの移行も簡単です。

Siriショートカット

ショートカットアプリや共有メニューからマンガを素早く登録できます。

対応プラットフォーム

iPhone / iPad / Mac / Apple Vision Pro

プライバシーポリシー

「マンガ曜日」は、ユーザーのプライバシーを尊重し、以下の方針でデータを取り扱います。

収集するデータ: 本アプリは個人情報を収集しません。ユーザーが登録したマンガデータ（作品名、URL、画像等）はユーザー自身のiCloudアカウントに保存され、開発者がアクセスすることはできません。

iCloud同期: データの同期にはAppleのCloudKitを使用します。データはユーザーのiCloudアカウントのプライベートデータベースに保存され、ユーザー本人のみがアクセスできます。

第三者への提供: ユーザーのデータを第三者に提供・販売することはありません。

分析・トラッキング: 本アプリは分析ツールやトラッキングツールを使用しません。

お問い合わせ: プライバシーに関するお問い合わせは GitHub Issues までお願いします。
